Review CSS

CSS Syntax

 1. Selector ตำแหน่งที่เลือกว่าจะกำหนดลักษณะต่างๆ
2. Declaration การประกาศลักษณะ
2.1 Property คุณสมบัติ
2.2 Value ค่าที่กำหนด

รูปแบบการเขียน

 1. Inline CSS เขียนภายใน tag HTML
2. Internal CSS เขียนภายใน Element HTML
3. External CSS นำเข้าไฟล์ CSS จากภายนอก

Selector

 1. ID ชื่อของ element โดยจะมีได้แค่ element เดียวเท่านั้น
2. Class ชื่อของ element โดยสามารถมีได้หลาย element
3. Pseudo Element การกำหนดเฉพาะบางส่วนของ element
4. Pseudo Class การกำหนดเฉพาะบางสถานะของ class
5. Combinators การกำหนดเงื่อนระหว่าง 2 selector
6. Attribute Selector การกำหนดการเลือกชื่อ attribute ใน element

Combinators

 1. Descendent Selector (space) a b = tag b ทั้งหมดที่อยู่ใน a
2. Child Selector (>) a > b = tag b ที่เป็นลูกของ a
3. Adjacent Sibling Selector (+) a + b = tag b ที่อยู่ล่าง a และติดกับ a เท่านั้น
4. General Sibling Selector (~) a ~ b = tag b ที่อยู่ล่าง a ทุก tag
